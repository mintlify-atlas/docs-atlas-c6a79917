# WooCommerce hooks and filters

WooCommerce provides hundreds of actions (hooks) and filters that allow you to modify and extend its functionality without editing core files. This guide covers the most important hooks and how to use them effectively.

## Understanding hooks and filters

### Actions (Hooks)

Actions allow you to execute custom code at specific points in WooCommerce's execution. They don't return values.

```php
// Example: Add custom content after the product title
add_action( 'woocommerce_single_product_summary', 'add_custom_product_content', 6 );

function add_custom_product_content() {
    echo '<div class="custom-content">Custom content here</div>';
}
```

### Filters

Filters allow you to modify data before it's used or displayed. They always return a value.

```php
// Example: Modify the product price
add_filter( 'woocommerce_product_get_price', 'modify_product_price', 10, 2 );

function modify_product_price( $price, $product ) {
    // Add 10% markup
    return $price * 1.1;
}
```

// Individual product in loop

## Checkout hooks

### Checkout fields

```php
// Add custom checkout field
add_filter( 'woocommerce_checkout_fields', 'add_custom_checkout_field' );

function add_custom_checkout_field( $fields ) {
    $fields['billing']['billing_company_vat'] = array(
        'type'        => 'text',
        'label'       => __( 'VAT Number', 'woocommerce-myextension' ),
        'placeholder' => __( 'Enter your VAT number', 'woocommerce-myextension' ),
        'required'    => false,
        'class'       => array( 'form-row-wide' ),
        'clear'       => true,
    );
    return $fields;
}

// Validate custom field
add_action( 'woocommerce_checkout_process', 'validate_custom_checkout_field' );

function validate_custom_checkout_field() {
    if ( isset( $_POST['billing_company_vat'] ) && ! empty( $_POST['billing_company'] ) ) {
        $vat = sanitize_text_field( $_POST['billing_company_vat'] );
        
        if ( empty( $vat ) ) {
            wc_add_notice( 
                __( 'Please enter a VAT number for your company.', 'woocommerce-myextension' ), 
                'error' 
            );
        }
    }
}

// Save custom field
add_action( 'woocommerce_checkout_update_order_meta', 'save_custom_checkout_field' );

function save_custom_checkout_field( $order_id ) {
    if ( isset( $_POST['billing_company_vat'] ) ) {
        update_post_meta( 
            $order_id, 
            '_billing_company_vat', 
            sanitize_text_field( $_POST['billing_company_vat'] ) 
        );
    }
}
```

## Hook naming conventions

WooCommerce follows these naming conventions from the [WordPress documentation standards](https://make.wordpress.org/core/handbook/best-practices/inline-documentation-standards/php/#4-hooks-actions-and-filters):

### Lifecycle hooks

Come in pairs ('before' and 'after'):

```php
do_action( 'woocommerce_before_calculate_totals', $cart );
// ... calculation logic ...
do_action( 'woocommerce_after_calculate_totals', $cart );
```

### Escape hooks

Allow short-circuiting of functions:

```php
$pre = apply_filters( 'pre_woocommerce_get_product', null, $product_id );
if ( null !== $pre ) {
    return $pre;
}
// ... continue with normal logic ...
```

## Best practices for using hooks

<Note>
Always use proper hook priorities to ensure your code runs in the correct order.
</Note>

### Priority and accepted arguments

```php
// Default priority is 10
add_action( 'woocommerce_before_cart', 'my_function', 10 );

// Run before default (lower number)
add_action( 'woocommerce_before_cart', 'my_early_function', 5 );

// Run after default (higher number)
add_action( 'woocommerce_before_cart', 'my_late_function', 20 );

// Specify number of accepted arguments
add_filter( 'woocommerce_product_get_price', 'my_price_function', 10, 2 );

function my_price_function( $price, $product ) {
    // Both parameters are available
    return $price;
}
```

### Removing hooks

```php
// Remove an action
remove_action( 'woocommerce_before_shop_loop', 'woocommerce_result_count', 20 );

// Remove a filter
remove_filter( 'woocommerce_product_get_price', 'my_price_function', 10 );

// Remove a hook from a class method
remove_action( 'woocommerce_checkout_order_processed', array( $class_instance, 'method_name' ), 10 );
```

<Warning>
When removing hooks, make sure to use the exact same priority and function name/reference as when the hook was added.
</Warning>

### Conditional execution

```php
add_action( 'woocommerce_before_cart', 'my_conditional_function' );

function my_conditional_function() {
    // Only run for logged-in users
    if ( ! is_user_logged_in() ) {
        return;
    }
    
    // Only run on certain days
    if ( date( 'N' ) >= 6 ) { // Saturday or Sunday
        wc_print_notice( __( 'Weekend special offers available!', 'woocommerce-myextension' ), 'notice' );
    }
}
```

### Always return filtered values

```php
// Good: Always return a value
add_filter( 'woocommerce_product_get_price', 'modify_price', 10, 2 );

function modify_price( $price, $product ) {
    if ( some_condition() ) {
        return $price * 1.1;
    }
    return $price; // Always return
}

// Bad: May not return a value
add_filter( 'woocommerce_product_get_price', 'bad_modify_price', 10, 2 );

function bad_modify_price( $price, $product ) {
    if ( some_condition() ) {
        return $price * 1.1;
    }
    // Missing return statement!
}
```

## Common use cases

### Add custom product badges

```php
add_action( 'woocommerce_before_shop_loop_item_title', 'add_custom_badges', 10 );

function add_custom_badges() {
    global $product;
    
    // New product badge (products added in last 30 days)
    $created = strtotime( $product->get_date_created() );
    if ( $created > strtotime( '-30 days' ) ) {
        echo '<span class="badge badge-new">' . __( 'New', 'woocommerce-myextension' ) . '</span>';
    }
    
    // Low stock badge
    if ( $product->managing_stock() && $product->get_stock_quantity() <= 5 ) {
        echo '<span class="badge badge-low-stock">' . __( 'Low Stock', 'woocommerce-myextension' ) . '</span>';
    }
}
```

### Customize email templates

```php
// Add custom content to order emails
add_action( 'woocommerce_email_order_meta', 'add_order_email_instructions', 10, 4 );

function add_order_email_instructions( $order, $sent_to_admin, $plain_text, $email ) {
    // Only for customer emails
    if ( $sent_to_admin ) {
        return;
    }
    
    // Add shipping instructions for specific shipping method
    $shipping_methods = $order->get_shipping_methods();
    foreach ( $shipping_methods as $shipping_method ) {
        if ( 'local_pickup' === $shipping_method->get_method_id() ) {
            echo '<h2>' . __( 'Pickup Instructions', 'woocommerce-myextension' ) . '</h2>';
            echo '<p>' . __( 'Please bring a valid ID when picking up your order.', 'woocommerce-myextension' ) . '</p>';
        }
    }
}
```
