## What is WooCommerce?

WooCommerce is a customizable, open-source ecommerce platform built on WordPress. It powers millions of online stores worldwide and provides a robust framework for developers to create extensions, themes, and custom integrations.

## What You Can Build

WooCommerce's flexible architecture enables developers to:

### Extensions

- **Payment Gateways**: Integrate custom payment methods using the Payment Gateway API
- **Shipping Methods**: Add custom shipping options and integrations
- **Product Add-ons**: Extend product functionality with custom fields and options
- **Marketing Tools**: Build integrations for email marketing, analytics, and more
- **Inventory Management**: Create custom inventory and warehouse management solutions

### Themes

- **Custom Storefronts**: Design unique shopping experiences
- **Classic Themes**: Build with traditional PHP templates and hooks
- **Block Themes**: Create modern FSE (Full Site Editing) themes
- **Mobile-Optimized Designs**: Responsive themes for all devices

### Integrations

- **Headless Commerce**: Build decoupled frontends using the Store API
- **Third-Party Services**: Connect to CRMs, ERPs, and other business tools
- **Custom Analytics**: Create specialized reporting and data visualization
- **Multi-Channel Sales**: Integrate with marketplaces and sales channels

## Architecture Overview

WooCommerce is built as a WordPress plugin with a modern monorepo structure:

```
woocommerce/
├── plugins/
│   └── woocommerce/          # Core plugin
│       ├── src/              # Modern PHP (PSR-4)
│       ├── includes/         # Legacy code
│       └── client/           # React admin
├── packages/
│   ├── js/                   # JavaScript packages
│   └── php/                  # PHP packages
└── tools/                    # Development tools
```

### Key Technologies

- **Backend**: PHP 7.4+, WordPress hooks system, PSR-4 autoloading
- **Frontend**: React, TypeScript, Gutenberg blocks
- **APIs**: REST API, Store API, PHP class APIs
- **Build Tools**: PNPM, Webpack, Composer

## Getting Started

<Steps>
  <Step title="Set up your development environment">
    Install Node.js, PNPM, PHP, and Composer. Clone the WooCommerce monorepo.
    
    ```bash
    git clone https://github.com/woocommerce/woocommerce.git
    cd woocommerce
    nvm install
    pnpm install
    ```
  </Step>
  
  <Step title="Build the project">
    Build all packages and plugins in the monorepo.
    
    ```bash
    pnpm build
    ```
  </Step>
  
  <Step title="Choose your path">
    Decide what you want to build:
    - [Extension development](/extensions/getting-started)
    - [Theme development](/themes/overview)
    - [API integration](/api/rest-api/overview)
    - [Core contribution](/contributing/overview)
  </Step>
</Steps>
