  <h2 className="text-2xl sm:text-3xl font-semibold text-gray-900 dark:text-white mb-4">
    Quick Start
  </h2>
  <p className="text-base text-gray-600 dark:text-gray-400 mb-8">
    Get up and running with WooCommerce development in minutes.
  </p>
  
  <Steps>
    <Step title="Set up your development environment">
      Install the prerequisites: Node.js (via NVM), PNPM, PHP 7.4+, and Composer. Clone the WooCommerce monorepo and run `pnpm install` to install dependencies.
      
      ```bash
      git clone https://github.com/woocommerce/woocommerce.git
      cd woocommerce
      nvm install
      pnpm install
      pnpm build
      ```
    </Step>
    
    <Step title="Explore the monorepo structure">
      The WooCommerce monorepo contains the core plugin, JavaScript packages, PHP packages, and development tools. Navigate to `plugins/woocommerce/` for the core plugin code.
      
      <Note>
        Modern code lives in `src/` with PSR-4 autoloading and dependency injection. Legacy code is in `includes/`.
      </Note>
    </Step>
    
    <Step title="Start building">
      Create your first extension, customize a theme, or integrate with the REST API. Check out our comprehensive guides to learn how to extend WooCommerce.
      
      ```php
      // Example: Add a custom action hook
      add_action('woocommerce_after_checkout_validation', function($data, $errors) {
          // Your custom validation logic
      }, 10, 2);
      ```
    </Step>
  </Steps>
</div>
